## 🚀 **HOW TO USE**

### **For Admin:**

**1. Access Admin Panel**
```
Route: /admin/support-agents
```

**2. Create Support Agent**
- Click "Create Agent" button
- Fill in details:
  - Full Name
  - Email (unique)
  - Password
  - Department
  - Role (Agent/Supervisor/Manager)
  - Max Concurrent Chats
- Click "Create Agent"
- Agent can now login

**3. Manage Agents**
- **Edit**: Click edit icon to update details
- **Reset Password**: Click lock icon, enter new password
- **Activate**: Click green check icon
- **Deactivate**: Click orange power icon
- **Suspend**: Click red user-x icon
- **Delete**: Click trash icon (with confirmation)

**4. Monitor Performance**
```
Route: /admin/support/analytics
```
- View real-time statistics
- Compare agent performance
- Export reports
- Filter by date range

---

### **For Support Agents:**

**1. Login to Support Panel**
```
Route: /support/login

Demo Credentials:
Email: support@houserentbd.com
Password: support123
```

**2. Access Dashboard**
After login, you'll see:
- Chat inbox on left
- Daily stats at top
- Chat area in center
- Notifications icon

**3. Handle Chats**
- Click on a chat in inbox to open
- Read customer message
- Type reply in message box
- Press Enter or click Send
- Mark as resolved when done

**4. Manage Profile**
- Click profile icon → My Profile
- Update your information
- Change password
- Upload profile photo

---

### **For Customers:**

**1. Start Chat**
- Click floating chat button on website
- Enter name and email
- Click "Start Chat"
- Wait for agent

**2. Chat with Agent**
- Send messages
- Receive instant replies
- Upload files (if needed)
- Rate experience when done
