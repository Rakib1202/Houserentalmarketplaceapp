│                       TECHNICAL DETAILS                                  │
└─────────────────────────────────────────────────────────────────────────┘

📁 FILES:
   - /components/LiveSupport.tsx            → User Live Chat Widget
   - /components/support/LiveChatReplyDashboard.tsx → Support Dashboard

🔌 API ENDPOINTS:
   - POST /api/support-tickets              → Create ticket (start chat)
   - POST /api/support-tickets/:id/messages → Add message
   - GET /api/support-tickets/:id           → Get ticket (polling)
   - GET /api/support-tickets               → Get all tickets (dashboard)
   - POST /api/support-tickets/:id/assign   → Assign to employee
   - PATCH /api/support-tickets/:id/status  → Update status

⚡ REAL-TIME:
   - User polls every 2 seconds for new support messages
   - Support polls every 3 seconds for new customer messages
   - Toast notifications for new messages

🗄️ DATABASE:
   - Collection: supportTickets
   - Each live chat = 1 support ticket
   - Messages array stores conversation history
